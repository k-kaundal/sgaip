Getting Started

Installation and first steps

Installation

Choose your preferred language and follow the installation guide below.

Python Installation

Step 1: Install the Package

Install SGAIP using pip:

pip install sgaip

Step 2: Verify Installation

Check that SGAIP is installed correctly:

sgaip --version

Step 3: Generate Your First Identity

Create a new cryptographic key pair:

sgaip keygen

This generates two files:

  • agent.sk - Your private key (keep this secret!)
  • agent.pk - Your public key (share this safely)

Step 4: Sign a Message

Create a cryptographic signature:

sgaip sign --private agent.sk --message "Hello, SGAIP!"

This creates a signature.bin file containing your digital signature.

Step 5: Verify a Signature

Verify that a signature is authentic:

sgaip verify --public agent.pk --signature signature.bin --message "Hello, SGAIP!"

If the message is correct, verification succeeds. If someone changes the message, verification fails.

Using SGAIP as a Library

You can also use SGAIP in your Python code:

from sgaip import SGAIP # Create an SGAIP instance sgaip = SGAIP() # Generate keys sgaip.keygen() # Your identity is derived from your public key identity = sgaip.get_identity() print(f"Your identity: {identity}") # Sign a message signature = sgaip.sign("Hello, world!") # Verify a signature is_valid = sgaip.verify(signature, "Hello, world!") print(f"Signature valid: {is_valid}")

Install SGAIP using npm:

Or add it to your project:

Using SGAIP in TypeScript

You can also use SGAIP in your TypeScript/JavaScript code:

import { SGAIP } from 'sgaip'; // Create an SGAIP instance const sgaip = new SGAIP(); // Generate keys const { publicKey, privateKey } = await sgaip.keygen(); // Your identity is derived from your public key const identity = sgaip.getIdentity(publicKey); console.log(`Your identity: ${identity}`); // Sign a message const message = 'Hello, world!'; const signature = await sgaip.sign(privateKey, message); // Verify a signature const isValid = await sgaip.verify(publicKey, signature, message); console.log(`Signature valid: ${isValid}`);

Common Patterns

Creating a New Agent Identity

Scenario: You want to create a new agent that can be identified and trusted.

# Generate a new key pair sgaip keygen --output agent # Publish your public key somewhere safe cat agent.pk # Keep your private key secure # Never share agent.sk with anyone

Signing a Document

Scenario: You want to sign a document to prove you created it.

# Create your document echo "This is my important document" > document.txt # Sign it with your private key sgaip sign --private agent.sk --message "$(cat document.txt)" > document.sig # Anyone can verify it with your public key sgaip verify --public agent.pk --message "$(cat document.txt)" --signature document.sig

Verifying Agent Identity

Scenario: You want to verify that a message came from a specific agent.

# Get their public key (from a trusted source) # Receive a message and signature from them # Verify the signature sgaip verify --public their_agent.pk \ --message "Their message" \ --signature message.sig

Troubleshooting

Command not found: sgaip

Make sure SGAIP is installed:

  • Python: Run pip install sgaip
  • JavaScript: Run npm install -g sgaip

Permission denied: agent.sk

Your private key file needs proper permissions:

chmod 600 agent.sk

Signature verification failed

Common causes:

  • The message text changed (even whitespace matters)
  • Using the wrong public key
  • The signature file is corrupted
