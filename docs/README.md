# GitHub Pages for SGAIP

This `/docs` folder contains the complete SGAIP website configured for GitHub Pages deployment.

## Quick Deployment

1. Go to **Repository Settings → Pages**
2. Select **Branch:** main  
3. Select **Folder:** /docs
4. Click Save

Your site will be live at: `https://k-kaundal.github.io/sgaip/`

## Files

- **index.html** - Main landing page
- **about.html** - About SGAIP
- **assets/css/style.css** - All styling
- **_config.yml** - GitHub Pages configuration

All other pages are available in the `/site` folder and can be copied here as needed.

## Setup Complete ✅

The website is ready to deploy!
