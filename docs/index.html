SGAIP

Stateless Global Agent Identity Protocol

Cryptographic identity for humans and AI agents without registries, blockchains, or centralized authorities

Get Started View on GitHub

Why SGAIP?

SGAIP defines identity as a cryptographic proof, not a registry entry, ledger record, or platform account.

Quick Start

Python Package

Install from PyPI and start using SGAIP in seconds:

pip install sgaip sgaip keygen sgaip sign --private agent.sk --message "hello" sgaip verify --public agent.pk --signature signature.bin --message "hello"
