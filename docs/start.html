Getting Started with SGAIP

Install, configure, and start using SGAIP in minutes

üêç Python Installation & Guide

Step 1: Install SGAIP

pip install sgaip

This installs the SGAIP package from PyPI with both CLI tools and library support.

Step 2: Verify Installation

sgaip --version

Confirms that SGAIP is properly installed and ready to use.

Step 3: Generate Your First Keys

sgaip keygen

Creates two files:

  • agent.sk ‚Äî Your private key (keep secret!)
  • agent.pk ‚Äî Your public key (safe to share)

Step 4: Sign a Message

sgaip sign --private agent.sk --message "Hello SGAIP"

Creates a cryptographic signature proving you authored the message.

Step 5: Verify a Signature

sgaip verify --public agent.pk --signature signature.bin --message "Hello SGAIP"

Confirms the signature is authentic without needing the private key.

Using SGAIP as a Python Library

from sgaip import SGAIP sgaip = SGAIP() sgaip.keygen() identity = sgaip.get_identity() signature = sgaip.sign("Hello world") is_valid = sgaip.verify(signature, "Hello world")

üöÄ JavaScript/TypeScript Installation & Guide

Step 1: Install SGAIP

npm install -g sgaip

For project use: npm install sgaip

Step 2: Verify Installation

sgaip --version

Confirms proper installation of the SGAIP CLI.

Step 3: Generate Keys

sgaip keygen

Creates agent.sk and agent.pk files.

Step 4: Sign & Verify

sgaip sign --private agent.sk --message "Hello SGAIP" sgaip verify --public agent.pk --signature signature.bin --message "Hello SGAIP"

Using SGAIP in TypeScript

import { SGAIP } from 'sgaip'; const sgaip = new SGAIP(); const { publicKey, privateKey } = await sgaip.keygen(); const identity = sgaip.getIdentity(publicKey); const signature = await sgaip.sign(privateKey, "Hello"); const isValid = await sgaip.verify(publicKey, signature, "Hello");

üîß Troubleshooting

Command not found: sgaip

Make sure SGAIP is installed in your PATH. Try:

python -m sgaip --version # Python npm list -g sgaip # npm

Permission denied on keys

Fix private key file permissions:

chmod 600 agent.sk

Signature verification failed

Common causes:

  • Message text changed (whitespace matters!)
  • Using wrong public key
  • Signature file corrupted
