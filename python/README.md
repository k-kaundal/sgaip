SGAIP Python reference package

Package: sgaip
Version: 0.1.0
Description: Stateless Global Agent Identity Protocol (SGAIP) reference implementation
License: Apache-2.0

Quick start

1. Clone the repository:

   git clone https://github.com/k-kaundal/sgaip.git

2. Change into the Python reference directory and build/install locally:

   cd sgaip/reference/python
   python -m pip install --upgrade build pip
   python -m build
   python -m pip install dist/*.whl
